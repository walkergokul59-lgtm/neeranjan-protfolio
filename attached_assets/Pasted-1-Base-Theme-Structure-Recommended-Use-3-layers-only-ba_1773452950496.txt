1. Base Theme Structure (Recommended)

Use 3 layers only: background, surface, accent.

Purpose	Color	Hex
Main background	Soft retro beige	#F2EFE7
Card background	Deep charcoal	#111111
Primary accent	Tech yellow	#FFD400
Secondary accent	Electric blue	#4F8CFF
Neutral text	Light gray	#D1D1D1
Heading text	Pure white	#FFFFFF

This removes harsh color clashes.

3. Fix Hero Section Contrast

Currently yellow text on black is good, but the small subtitle is hard to read.

Improve it like this:

Heading:
WHITE (#FFFFFF)

Highlight word:
YELLOW (#FFD400)

Subtitle:
GRAY (#A0A0A0)

Example:

TURNING IDEAS INTO
INTELLIGENT SYSTEMS

Where INTELLIGENT SYSTEMS = yellow.

8. Typography Color Hierarchy

Follow this rule everywhere:

Element	Color
Headings	#FFFFFF
Highlight	#FFD400
Normal text	#D1D1D1

9. Pro-Level Color Rule

Use 80 / 15 / 5 rule

80% neutral (black / gray)
15% primary (yellow)
5% accent (blue / purple / teal)

Your current site is more like 40% bright colors, which hurts readability.
